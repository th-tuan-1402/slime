# Slime - Hanbai Migration

Headless API backend (Laravel 11 + PostgreSQL + Redis) and Nuxt 3 frontend (Tailwind + Nuxt UI).

## Prerequisites

- Docker & Docker Compose

## Quick Start

```bash
# 1. Create the Laravel project (first time only)
docker run --rm -v "$(pwd):/app" -w /app composer create-project laravel/laravel backend

# 2. Build and start all services
make build
make up

# 3. Run database migrations
make migrate
```

The API will be available at `http://localhost:8080`, and the client at `http://localhost:3000` when running via Docker.

## Services

| Service       | Container     | Port  |
|---------------|---------------|-------|
| App (PHP)     | slime-app     | 8080  |
| Client (Nuxt) | slime-client  | 3000  |
| PostgreSQL    | slime-db      | 5432  |
| Redis         | slime-redis   | 6379  |

## Common Commands

```bash
make up                # Start services
make down              # Stop services
make build             # Rebuild containers
make shell             # Open bash in app container
make test              # Run tests
make test-coverage     # Run tests with coverage
make phpstan           # Run static analysis
make migrate           # Run migrations
make fresh             # Fresh migrate + seed
make seed              # Run seeders
make logs              # Tail logs
make ps                # Show container status
make install           # Install composer dependencies
make openapi-export    # Export OpenAPI spec
make openapi-validate  # Export and validate OpenAPI spec
make ci                # Run full CI locally
make client-dev        # Run Nuxt dev server (from host)
make client-build      # Build Nuxt for production
```

## Environment

Configuration is in `backend/.env`. Key settings:

- **DB**: PostgreSQL 15 (`db:5432`, database: `slime`)
- **Cache/Queue/Session**: Redis 7 (`redis:6379`)

## OpenAPI Documentation

The API specification is auto-generated by [Scramble](https://github.com/dedoc/scramble) from the source code.

| Endpoint | Description |
|----------|-------------|
| `/docs/api` | Swagger UI viewer |
| `/docs/api.json` | Raw OpenAPI 3.x JSON spec |

```bash
make openapi-export      # Export spec to openapi.json
make openapi-validate    # Export and validate with Redocly
```

## Static Analysis

PHPStan with Larastan at level 8. Run via:

```bash
make phpstan
```

## CI Pipeline

GitHub Actions runs on every push/PR to `main`:

1. **PHPStan** — static analysis at level 8
2. **PHPUnit** — full test suite
3. **OpenAPI Validate** — export and lint the OpenAPI spec

```bash
make ci    # Run full CI locally (phpstan + test + openapi-validate)
```
